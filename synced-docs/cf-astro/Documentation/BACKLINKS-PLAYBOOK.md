# Backlinks & Citations Playbook

Owner-facing playbook for off-page SEO — the one lever that cannot be coded.
Backlinks (links from other sites to madagascarhotelags.com) and brand
mentions are the strongest remaining ranking input for this site: they feed
classic Google rankings, the local pack, and AI-engine citations (2026
research: consistent third-party brand mentions correlate with being cited
by ChatGPT/Perplexity/AI Overviews even more strongly than raw link counts).

Companion to [SEO-OPERATIONS.md](./SEO-OPERATIONS.md) §5–6. Re-check
quarterly.

---

## 1. Goals & KPIs

- **2–3 earned links or listings per quarter** (quality over volume — one
  local vet or newspaper link beats fifty directory spam links).
- Track monthly: referring domains (Bing Webmaster Tools → Backlinks, free),
  local-pack position for "hotel para perros aguascalientes" / "pensión
  canina aguascalientes", and GSC branded-query impressions.
- Never buy links, join link farms, or exchange reciprocal-link pages —
  Google penalizes both sides.

### Tier 2 — partnerships (ongoing, highest local value)

- **Veterinarias** near both branches (Héroes / Jardines del Sol):
  cross-referral agreement + "hospedaje recomendado" mention on their
  site/Facebook. Offer the same back from the blog.
- **Asociaciones protectoras / rescates de Aguascalientes**: sponsor or host
  an adoption day — generates press, social mentions and links naturally.
- **Groomers, pet shops, entrenadores** that don't offer boarding:
  complementary-service referral exchange.

### Tier 3 — earned media (2–3 pitches per quarter)

- **LJA.mx** and **El Heraldo de Aguascalientes**: pitch listicle/seasonal
  angles — "mejores hoteles para perros en Aguascalientes", "qué hacer con
  tu mascota en vacaciones", Semana Santa / verano / diciembre.
- **Feria de San Marcos**: pet-friendly visitor roundups — the site's
  travel-with-pets blog post is the natural citation.
- Local podcasts/YouTube (Aguascalientes lifestyle) — 30 years of history
  (since 1994) is a story hook.

## 7. Owner checklist (one-time, cannot be done from code)

- [ ] 🚨 **URGENT — turn OFF Cloudflare Managed robots.txt** (live-verified ON
      2026-07-18, blocking GPTBot/ClaudeBot/CCBot/Google-Extended and more
      with `Disallow: /`): Cloudflare dashboard → zone → **AI Crawl Control**
      (or Security → Bots) → "Manage AI bots robots.txt" → **OFF**, and set
      each AI crawler listed in SEO-OPERATIONS.md §1.1 to **Allow**. Verify:
      `https://madagascarhotelags.com/robots.txt` must start with the
      "Madagascar Pet Hotel" comment — no "BEGIN Cloudflare Managed content".
- [ ] **Workers Builds deploy command**: Workers & Pages → cf-astro →
      Settings → Build → Deploy command:
      `npx wrangler deploy && node scripts/indexnow-ping.mjs`; also turn ON
      Caching → Configuration → Crawler Hints.
- [ ] **Bing Webmaster Tools**: register at bing.com/webmasters ("Import
      from Google Search Console"), then paste the `msvalidate.01` code into
      the commented placeholder in `src/layouts/BaseLayout.astro` (search
      for `msvalidate`). Bing powers ChatGPT search citations (~87% overlap).
- [ ] **Verify the X/Twitter handle**: `BaseLayout.astro` emits
      `twitter:site: @hotelmadagascar` — confirm that account exists and is
      yours, or remove the tag.
- [ ] **GBP, Cloudflare AI Crawl Control, GSC**: follow
      [SEO-OPERATIONS.md](./SEO-OPERATIONS.md) §1–4.
- [ ] After each deploy that fixes GSC-reported URLs, press **Validate
      Fix** in the matching GSC report (see SEO-OPERATIONS.md §2a triage
      table).
